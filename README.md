## Task Management
### Author Ehsan

Laravel task management

- follow steps by order

1.
```
git clone https://github.com/essejblack/task-management.git
```
2.
```
composer install
```
3.
```
php artisan db
create database taskmanagement
```
4.
```
php artisan migrate --seed
```
5.
```
php artisan serve
```
6.
```
open http://127.0.0.1:8000 in your browser
```
